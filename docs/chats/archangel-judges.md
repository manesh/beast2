## moderation language

Judges are not priests.

Judges are process guardians.

A judge may rule on:

- spam
- duplicate claims
- misclassified evidence
- forged evidence
- harassment
- bad-faith summaries
- unverifiable testimony labels
- public figure misattribution

Every ruling should have:

- judge name
- timestamp
- reason
- affected content
- appeal path

No hidden court.

No anonymous moderation.
