# Warp Speed Pirate (WaSP)
Version: 0.1 Prototype Design

## High Concept

A 2D pixel-art space piracy game focused on boarding actions, crew management, and ship scavenging.

The player is an AI captain aboard the WaSP (Warp Speed Pirate).

The ship is built from stolen modules taken from defeated vessels.

Primary gameplay loop:

1. Find target
2. Board target
3. Defeat crew
4. Capture/recruit survivors
5. Steal ship modules
6. Expand ship
7. Repeat

# Core Gameplay Pillars

## Boarding Combat

Primary focus of the game.

Ship-to-ship combat is secondary.

Most battles are resolved by boarding.

Features:

- melee combat
- ranged weapons
- breaching charges
- corridor fights
- room clearing
- reactor sabotage
- hostage rescue/capture
- captain duels

---

## Ship Salvaging

Enemy ships are loot containers.

After victory, player may steal:

- engines
- reactors
- cargo bays
- med bays
- workshops
- prisons
- shield generators
- hydroponics
- weapons rooms

Stolen modules become part of the player's ship.
