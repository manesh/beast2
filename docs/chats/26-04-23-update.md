I’d standardize them like this:

beast-canonical-sdxl-base-2026-04#sha256:<hash>

user-bundle-sdxl-base-unet-swap-user-style#user:<name>

user-bundle-sdxl-base-unet-swap-user-style#sha256:<hash>

author: userName

change: unet swapped

name = human readable
hash = identity
metadata = story

beast-canonical-sdxl-base-2026-04

is the label.

#sha256:...

is the exact identity.

derived-from:
changed:
author:
