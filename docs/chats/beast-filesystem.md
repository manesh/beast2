files and file paths beast2

beast two does not try to hide the reality of file systems.

Beast two places a crumb file in directories it places other files in. The crumb file always begins with a hyphen. The main purpose of crumb files is to link file directories on slow storage to databases on solid state storage.

in the beast two app there is an interface for adding directories to beast twos using a management interface.

in addition to using multiple directories the Beast 2 app less user choose beautiful file names formatted as they see fit this improves interoperability and readability when browsing file folders from other apps. beast2 is designed to be as open as possible allowing maximum apps to be used in the ecosystem.
