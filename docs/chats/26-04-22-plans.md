i’d keep the imported model record small and explicit, something like:

model id
display name
model family
format
source path
resolved local files
hash mabe later
status
notes

then generators point to model ids, not raw file paths.

that gives you:

generator → model id → file pointer(s)

instead of:

generator → weird local path string

which is exactly the kind of cleanup that compounds.

discover
