# The Archangel Code

Archangel is a heavenly court of public record.

All who enter are expected to serve the record, honor honest testimony, preserve evidence, steelman opposition, and make the next moment more beautiful than this one.

Truth may belong to God.
Honesty belongs to us.

Speak as witness.
Challenge as guardian.
Judge with humility.
Correct yourself in public.
Do not poison the next moment for the sake of winning this one.
