**Generator: Hero Lifts the Villain (Meme Forge)**  
  
**Purpose**  
  
Create short, shareable meme videos showing a classic story arc:  
  
```
hero vs villain
villain defeated
hero offers hand
villain becomes ally

```
  
The generator combines:  
	•	cinematic meme generation  
	•	moral storytelling  
	•	LLM social simulations  
  
The result is a gallery of **betrayal memes** and **alliance memes** generated automatically from agent interactions.  
  
⸻

**LLM Simulation Engine**  
  
The meme generator is driven by an internal **role-play simulation**.  
  
Two LLM agents interact:  
  
```
Agent Hero
Agent Villain

```
  
Each round they choose:  
  
```
Cooperate
Betray

```
  
Similar to a simplified Prisoner’s Dilemma interaction.  
  
Outcome mapping:  
  
**Agent Behavior**	**Meme Generated**  
mutual cooperation	alliance meme  
one betrays	betrayal meme  
repeated betrayal	escalating chaos meme
